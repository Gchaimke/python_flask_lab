# python_flask_lab

simple flask starter project with SQLLite

# Instalation

add new environment, from project root run:
> py -3 -m venv venv
> venv\Scripts\activate
> pip install Flask

Install app
> pip install -e .

Initialize the Database File
> flask init-db

For test install:
> pip install pytest coverage

run program from index.wsgi or 
> flask run
